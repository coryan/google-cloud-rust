# Google Cloud Client Libraries for Rust - Well Known Types Conversions

This package contains helpers to convert between the well-known types in [google-cloud-wkt] and those in [prost-types].

**WARNING:** this crate is under active development. We expect multiple breaking
changes in the upcoming releases. Testing is also incomplete, we do **not**
recommend that you use this crate in production. We welcome feedback about the
APIs, documentation, missing features, bugs, etc.

[google-cloud-wkt]: https://crates.io/crates/google-cloud-wkt
[prost-types]: https://crates.io/crates/prost-types
